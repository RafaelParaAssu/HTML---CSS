Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito utilizando a tag B (não semântica).

Já nesta frase, temos um termo em destaque, dessa vez utilizando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, destacamos o termo em itálico usando a tag I (não semântica).

Assim como a característica em negrito, temos também nessa frase em itálico, um termo com ênfase, utilizando a tag EM (semântica).

Texto marcado

Podemos criar um texto marcado utilizando a tag MARK.

Podemos alterar a cor do texto marcado, utilizando o parametro style dentro da tag MARK, para destacar apenas esse texto, ou de forma geral em todos os textos marcados, aplicando o parâmetro style dento do cabeçalho HEAD.

Texto grande e texto pequeno

Quando criamos um texto grande e um texto pequeno nesse parágrafo, vimos que a tag BIG está obsoleta, porém a tag SMALL ainda está em uso, pois ela tem significado (semântica).

Texto Deletado / Inserido

Podemos marcar um texto como excluído para indeicar que ele deve ser lido, porém não considerado.

Em contrapartida, podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.

Para o texto excluído utilizamos a tag DEL, e para o texto inserido, utilizamos a tag INS.

Texto Sobescrito / Subscrito

Utilizamos o texto sobescrito para expressões do tipo 4X3+2X2=6X+0.

Já para o texto subscrito podemos utilizar como exemplo H3CO2CH3.

As tags utilizadas foram, tag SUP para indicar um texto sobescrito, e a tag SUB para indicar um texto subscrito.
